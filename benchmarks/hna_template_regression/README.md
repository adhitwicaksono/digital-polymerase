**Planned extension:** Add Prototype 001C / long-input testing after HNA scaling to 111-mer is completed

## Summary

Benchmark 005 documents the refactored RNA → HNA workflow in Digital Polymerase.

The original HNA converter was the first successful RNA → XNA proof of concept in this project, but it came from the early one-off prototype stage. After the ANA and FANA development arcs introduced stricter chain-continuity validation and clearer reporting, HNA was revisited as a regression/refactor benchmark.

Prototype 001B uses a **full-length HNA template** as the backbone/scaffold donor and transplants the exact RNA base sequence into the HNA framework.

Current tested case:

```text
RNA-8mer.pdb + 481d-HNA8nt.pdb → converted_hna_RNA-8mer_001B.pdb
```

The 8-mer output was visually inspected in PyMOL and Discovery Studio style views and appeared coherent as a short HNA-like candidate structure.

## Method

Prototype 001B performs **full-template-guided RNA → HNA reconstruction**.

For each source RNA residue:

1. select the corresponding HNA template residue by position
2. take the HNA backbone/scaffold atoms directly from the template
3. align RNA local sugar anchors to the HNA template residue
4. transform RNA nucleobase atoms into the HNA residue frame
5. combine the HNA scaffold with the transformed RNA base
6. write an HNA-like candidate residue
7. validate chain continuity and base-attachment distances

This differs from the ANA/FANA chain-preserving strategy because HNA is more structurally divergent from RNA. In this case, using the experimental HNA template as the scaffold donor is more appropriate for the short 8-mer test.

## Current Boundary

Prototype 001B is intentionally strict.

It requires the HNA template to be at least as long as the source RNA input.

For example, this is allowed:

```text
RNA 8-mer + HNA 8-mer template
```

This is not allowed by Prototype 001B:

```text
RNA 12-mer + HNA 8-mer template
```

The converter correctly stops in that situation, because recycling an 8-mer HNA template silently would overclaim the result.
